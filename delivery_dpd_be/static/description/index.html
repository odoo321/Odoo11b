DPD Delivery for BENELUX

This module allowes you to ship from Belgium, The netherlands and Luxemburg towards all other countries.

The API for DPD benelux is different from the other countries.

If you are shipping from other countries we can highly suggest the Delivery DPD website (by Steigend IT Solutions)

This module doesn't allow to ship towards parcelshops.

If you want also to make it possible for you customers to select a nearby parcelshop, then we can highly suggest to buy

2 in 1:Home and parcelshop delivery module.

Basic Backend Configuration

Go to menu Inventory -> Delivery Methods -> Configure the DPD Account Details

Backend Process

1. On sale order select delivery method, confirm the sale order, a picking will be created.

2. Confirm the picking and you can see the DPD Label will be generated in a downloadable
format.

3. Here is the sample DPD Label generated.

4. A new tab will be visible for delivery information

5. You can update the tracking information manually by pushing the button, or you can turn on the setting automatic synchronize setting on your delivery carrier.
